# 🏨 Hotel Booking App (Full Stack)

## 🚀 Live Demo

https://hotelbooking-murex-one.vercel.app/

📌 Description

A modern full-stack hotel booking application built using React, Node.js, and MongoDB.

✨ Features :-
Browse hotel rooms,
View room details,
Booking functionality,
Responsive design,

🛠 Tech Stack :-
Frontend,
React.js,
Tailwind CSS,
Backend,
Node.js,
Express.js,

Database :-
MongoDB

👨‍💻 My Role :-
Built frontend using React,
Integrated APIs,
Designed responsive UI,
Implemented booking flow,

🔗 Backend Repository :-
Backend is private and can be shared upon request.
